### Development

运行

1. 安装redis并启动
关于redis的安装与运行可以参考[这里](https://github.com/antbaobao/AntBlog/issues/42)
2. 安装mongodb并启动
3. 开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Deploy

```bash
$ npm start
$ npm stop
```
